Core

Virtual Lab for Robotics Students

CORE is an online platform for developing robotics reinforcement learning algorithms. Using OpenAI Gym libraries and an integrated code editor with syntax highlighting, CORE offers a seamless environment for students to code robots with minimal performance overhead.

Join Now

Built on Open Source Software

CORE is powered by open-source OpenAI Gym libraries, a widely used toolkit for developing reinforcement learning algorithms. Students develop and submit code within CORE, receiving feedback through generated robot simulation videos and terminal output.

Lightweight Development

All code compilation takes place in the cloud, giving users the flexibility to develop using any device, without needing to install local software.

Built for Students and Teachers

CORE enables instructors to manage virtual classrooms, track student progress, and review submitted code from each environment.

Optimized for All Devices

CORE is fully responsive and optimized for smartphones, tablets, and desktops, providing a smooth user experience across all platforms.
