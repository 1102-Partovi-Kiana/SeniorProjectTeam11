What is Given

  • global env: This represents the robot's environment, which contains the robot's current state and methods for interacting with it. The environment has various "getters" and "setters."
  • forward, right, left, backward: These values are retrieved using the robot's sensor functions and represent the distances to obstacles in the corresponding directions.
  • action: A list representing the robot‚Äôs movement and steering inputs.
  • values: A dictionary that maps sensor directions to their distance values.

What Needs to be Retrieved

  • forward: The sensor value representing the distance directly in front of the robot.
  • right: The sensor value representing the distance to the robot's right.
  • left: The sensor value representing the distance to the robot's left.
  • greatest_direction: The direction with the highest sensor value, indicating the most open space.
  • greatest_value: The corresponding sensor value of the greatest_direction.
