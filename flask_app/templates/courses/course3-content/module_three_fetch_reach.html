About the Fetch Robot Class

The Fetch robot class consists of multiple tasks the Fetch Robot can complete including, reaching, picking up and placing objects, stacking objects on top of each other, using sensors to organize certain objects, etc. These robots are great for students to learn about robotics, programming, and how robots move and complete tasks.

Fetch Reach

The Fetch Reach robot is designed to demonstrate basic reaching mechanics, teaching students how to program robotic arms to move towards objects efficiently.

Fetch Pick-and-Place

This Fetch Pick and Place robot focuses on object manipulation, including picking up and placing objects with precision. Students will learn how to program the robot to move objects from one location to another. This is essential in future tasks of organizing.

Fetch Organize

This Fetch Organize robot organizes objects based on pre-defined criteria. It is a wonderful robot in aiding students with the exploration of simulating everyday tasks. Fetch Organize with Sensors

With additional sensor capabilities, this Fetch Organize robot organizes objects based on pre-defined criteria. Students will learn how to program the robot to sort items by characteristics utilizing its built-in sensors. The robot can detect different objects based on what it "sees" or "senses."
