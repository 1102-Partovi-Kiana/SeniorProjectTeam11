About the Fetch Organize Robot

Learn the purpose of the Fetch Organize Robot.

Let's Explore

Fetch Organize Robot

The Fetch Organize robot is designed to use what students have previously learned about the robots with the integration of color based organization. The goal of this robot is to identify objects, grasp them, and place them in the correct container.

Fetch Organize Robot Gallery


The first image represents the simulated Fetch Organize robot in MuJoCo and OpenAI Gym, which you will be programming. This virtual environment allows you to experiment with the robotic and learning the its functionality in a safe and controlled setting.

The next three images showcase the real Fetch Organize robot, which operates in physical environments using the Robot Operating System (ROS).

Simulation vs. Real Fetch Organize Robot

While youâ€™ll be working with the simulated Fetch Organize robot in OpenAI Gym (an open-source toolkit for training robotic tasks) using MuJoCo (a physics engine), a real Fetch Organize robot exists, and the skills you learn here are transferable to real-world applications.

Aspect Simulated Fetch Organize Robot Real Fetch Organize Robot
Physics Simulated physics engine Real-world physics (friction, noise, delays)
Sensor Data Simulated sensors (perfect readings) Real sensors (cameras, force sensors, encoders)
Execution Speed Instant (fast simulation) Slower (real motors need time to move)
Communication Uses OpenAI Gym API Uses ROS (Robot Operating System)
Error Handling Can reset instantly Needs safety checks to avoid damage
Failure Risk No risk Hardware can break
