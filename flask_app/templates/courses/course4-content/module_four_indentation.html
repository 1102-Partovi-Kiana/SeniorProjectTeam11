Indentation

Learn proper indenting practice to further enhance the readability of your code.

Let's Dive In

Why Indentation Matters

Indentation improves the visual structure of the code, showing the relationship between different blocks like loops, conditions, or functions.

Most languages recommend a consistent use of spaces (usually 4 per level) or tabs, but mixing both is discouraged.

Proper indentation ensures the code is readable and less prone to logical errors caused by misaligned blocks.

# Good
def check_album_year(album, year):
if album == "1989" and year == 2014:
    return "Correct Match"
return "No Match"

# Bad
def check_album_year(album, year):
if album == "1989" and year == 2014:
return "Correct Match"
return "No Match"
