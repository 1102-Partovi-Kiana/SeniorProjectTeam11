Functions

Learn how to utilize functions to make your code modular.

Let's Dive In

What are Functions?

A function is a reusable block of code designed to perform a specific task. They follow the DRY principle: Don't Repeat Yourself. Use functions to avoid duplicating code

Functions are used to:

  • Encapsulate logic into reusable components.
  • Reduce code duplication.
  • Make programs easier to debug.
  • Make programs easier to maintain.

Function Syntax

In Python, a function is defined using the def keyword:

# Example of a simple function
def print_hello():
print("Hello, world!")

To call the function, simply use its name followed by parentheses:

print_hello()  # Outputs: Hello, world!

Function Parameters

Functions can accept parameters. Parameters are specified in parentheses during function definition:

# Function with parameters
def print_name(name):
print(f"Hello, {name}!")

When calling the function, provide the proper arguments for the parameters:

print_name("Bob")  # Outputs: Hello, Bob!

Return Values

Functions can return values using the return statement, allowing you to get and use the result:

# Function with a return value
def add_numbers(a, b):
    return a + b

result = add_numbers(4, 2)
print(result)  # Outputs: 6
