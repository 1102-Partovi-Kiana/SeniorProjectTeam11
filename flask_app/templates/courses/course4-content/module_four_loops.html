Loops

Grasp the depth of how loops work.

Let's Dive In

1. For Loops

For loops are ideal when you know when you know the number of iterations you want to complete in advance. They are commonly used to iterate over collections/sequences.

# Example: Iterate through a list of favorite albums
albums = ["1989", "Folklore", "Lover"]
for album in albums:
    print(f"Album: {album}")

This loop will print each album in the list.

2. While Loops

While loops are best suited for scenarios where the number of iterations depends on a condition that changes dynamically. They allow for more flexibility but require careful attention to ensure the condition eventually becomes false to avoid infinite loops.

# Example: Count down from 10
count = 10
while count > 0:
    print(f"Countdown: {count}")
    count -= 1

This loop will print a countdown until the value of count reaches 0.

6. Do-While Loops

Do-while loops guarantee that the loop body executes at least once, as the condition is checked after the loop executes. They are useful in cases where the initial execution of the loop body is necessary, such as displaying a menu repeatedly until the user chooses to exit.

# Python doesn't have a native do-while loop, but it can be simulated:
condition = False

while True:
    print("\nMenu:")
    print("1. Option 1")
    print("2. Option 2")
    print("3. Exit")
    
    choice = input("Enter your choice (1-3): ")
    
    if choice == "1":
        print("You chose Option 1.")
    elif choice == "2":
        print("You chose Option 2.")
    elif choice == "3":
        print("Exiting the menu. Goodbye!")
        break
    else:
        print("Invalid choice. Please try again.")

This loop simulates a do-while structure by displaying a menu that runs at least once.

5. Best Practices

  • Use descriptive variable names in your loops to make your code easier to follow along with.
  • Make sure there's a clear exit condition to avoid infinite loops.
