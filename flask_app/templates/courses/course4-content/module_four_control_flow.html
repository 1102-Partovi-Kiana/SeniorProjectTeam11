Control Flow

Understand how to guide your program's behavior with tools like conditionals, loops, and other control structures

Let's Dive In

1. Conditional Statements

Conditional statements allow your program to make decisions based on conditions.

# Example: Deciding which album to recommend with if-else statements
favorite_genre = "pop"

if favorite_genre == "pop":
    print("You might like Taylor Swift's '1989'")
elif favorite_genre == "rock":
    print("Check out 'Bohemian Rhapsody' by Queen")
else:
    print("Explore some new genres!")

2. Loops

Loops are used to repeat actions, either a specific number of times or until a condition is met. They follow the DRY principle: Don't Repeat Yourself. Use loops to avoid duplicating code.

# Example: Print top 3 albums
top_albums = ["1989", "Fearless", "Folklore"]

for album in top_albums:
    print(album)

# Example: Countdown to a concert
countdown = 5
while countdown > 0:
    print(f"{countdown} days left!")
    countdown -= 1

3. Break and Continue

Use break to exit a loop early or continue to skip the current iteration.

# Example: Stop when finding the favorite album
albums = ["Red", "1989", "Midnights", "Folklore"]

for album in albums:
    if album == "1989":
        print("Found the favorite album!")
        break

# Example: Skip specific albums
for album in albums:
    if album == "Midnights":
        continue
    print(album)

4. Nested Control Flow

Control flow statements can be nested to handle more complex logic.

# Example: Checking album ratings
album_ratings = {
    "1989": 9,
    "Fearless": 8,
    "Reputation": 7,
}

for album, rating in album_ratings.items():
    if rating > 8:
        print(f"{album} is highly rated!")
    else:
        print(f"{album} has a decent rating.")
