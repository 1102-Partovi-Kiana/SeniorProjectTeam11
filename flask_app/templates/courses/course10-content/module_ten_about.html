Fetch Robot with Sensors

The Fetch Sensors robot is designed to use what students have learned about the robots with the integration of sensors. Sensors allow the robot to detect objects, and measure distances for adaptive learning with its task.

Fetch Sensors Robot Simulation Video

Simulation vs. Real Fetch Sensors Robot

The Fetch Sensors robot exists both in simulation (MuJoCo, OpenAI Gym) and in real life. While the simulation helps with training, the real-world version comes with added complexity.

Aspect Simulated Real
Physics Simulated in MuJoCo Real-world friction, noise, and delays
Sensor Data Perfect, noise-free Uses depth cameras, LiDAR, force sensors (with noise)
Execution Speed Fast, near-instant Slower, depends on hardware
Communication OpenAI Gym API ROS (Robot Operating System)
Sensor Limitations Perfect vision, no interference Lighting, occlusions, calibration issues
Error Handling Instant reset Requires safety checks
Failure Risk No risk Motors, sensors can break
Learning Transfer Trains fast, but may not fully transfer Requires tuning for real-world conditions
